Parágrafos e quebras de linha (ex3)

Os parágrafos podem ser escritos de qualquer jeito, sejam pulando linhas ou identando o texto o resultado será o mesmo enquanto o escrito estiver dentro da tag <p>

SE houver a necessidade de quebra de linha
(como essa por exemplo), utilizamos a tag <br> (não utilizamos diversos <br> para pular diversas linhas, esse trabalho é do CSS e seu estilo de folhas em cascata)
